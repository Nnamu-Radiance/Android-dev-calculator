PROJECT TITLE
Dart Grade Calculator - Automated Student Grade Processing System

BRIEF DESCRIPTION
A Dart console application that automates the process of calculating student 
grades and grade point averages (GPA) from raw marks. The application reads 
student names and marks from an Excel file, applies a predefined grading scale, 
and exports the results to a formatted Excel file with comprehensive console 
output.

DETAILED DESCRIPTION
The Dart Grade Calculator is a command-line utility designed for educators and 
academic institutions to efficiently process large datasets of student marks 
and convert them into standardized grades. The application handles the complete 
workflow from data ingestion to result export, with built-in validation and 
error handling to ensure data integrity.

KEY FEATURES
• Excel File Processing: Read and write .xlsx files programmatically
• Automated Grade Calculation: Instantly convert marks to letter grades and GPA
• Data Validation: Validates marks are within the 0-100 range
• Formatted Output: Displays results in an organized table format
• Flexible File Handling: Accept custom input/output file paths via CLI
• Error Handling: Graceful handling of invalid data and missing files
• Scalable: Process 50+ students efficiently

TECHNICAL STACK
Language:      Dart (>=2.19.0)
Platform:      Cross-platform (Windows, macOS, Linux)
Key Libraries: archive, xml, args
File Format:   Microsoft Excel (.xlsx)

USAGE EXAMPLE

INPUT REQUIREMENTS
• Excel file with two columns minimum
• Column A: Student Names (text)
• Column B: Marks (numeric, 0-100 range)
• Header row in first row

OUTPUT GENERATED
• Console table displaying all processed students
• Excel file with four columns:
  - Student Name
  - Marks (original score)
  - Grade (letter grade)
  - GPA (grade point value)

SYSTEM REQUIREMENTS

• Supported on Windows, macOS, and Linux
